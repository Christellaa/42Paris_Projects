# 42Paris_Projects

This repository contains projects completed as part of my cursus at 42 School, a software engineering school known for its hands-on and project-based learning approach.

# About 42 School

42 School offers an intensive program with no classes or teachers, based entirely on:
- project-based learning
- self-directed study
- peer-to-peer collaboration
- real-world problem solving

The curriculum emphasizes autonomy, rigor, adaptability and continuous learning.

### Core Curriculum (2 years)
- Programming: C, C++, Web
- Algorithms, data structures
- Systems, networking
- Software architecture
- Teamwork, collaborative project management

### Specialization (3 years - ongoing)
- DevOps: Docker, CI/CD, monitoring, automation
- Cybersecurity
- AI applied to infrastructure and systems
